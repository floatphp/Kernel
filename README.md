# FloatPHP (Kernel)

<img src="https://floatphp.com/assets/img/floatphp.png" width="100" alt="Micro PHP framework">

FloatPHP **Kernel Components**.
Used to setup WEB application using all other components.

## ⚡ Installing:

```
composer require floatphp/kernel
```
